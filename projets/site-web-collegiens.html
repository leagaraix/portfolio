Développement d'un site web à destination de collégiens

Conception d’un site web pour une ESN (SCC), à destination des collégiens et facilitant la demande de stages. Nous avions pour modèle l'entreprise SCC.

Premier projet en équipe, première utilisation de Git : nous avons fait de nombreuses erreurs, nous nous sommes mal organisés... mais ça nous a bien servi pour les projets suivants !

Réalisations de l'équipe (4 personnes)

Recherches sur l’entreprise cliente : étude de ses caractéristiques, analyse de ses besoins

Conception des pages : création de la maquette, ajustements aux demandes du client

Développement : écriture du code HTML et CSS, corrections

Adaptation du contenu du site au public (collégiens) : textes simples, peu de vocabulaire spécifique, page dédiée à la demande de stage, contact simplifié, sélection des informations utiles pour la compréhension du métier de l’entreprise

Mes contributions

Conception de la maquette :

  • schéma sous forme d'une arborescence pour visualiser la hiérachisation du site
  • vérification de la cohérence entre les pages
  • ajout de commentaires pour faciliter la compréhension

Développement : rédaction de la majeure partie du code CSS et de la page Nous contacter
